실습설명

  1. 윈도우 탐색기에서 C:/dve 폴더를 생성한다. (프로젝트를 하위로 생성하기 위함. 이름은 각자 알아서...)
  2. "google에서 sublime text 를 검색하여 에디터를 설치"
    • project > add folder to project 메뉴를 이용하여 C:/dev 폴더를 추가한다.
  3. "goole 에서 github 를 검색하여 회원가입"
    • 소스 작성시 Copy & paste 금지!
    • 무료인 free로 가입을 한다.
    • "Repositary를 {username}.github.io로 생성한다. ("http://pages.github.com/")."
    • IInitialize this repository with a README 를 체크해준다.
    • Clone in Desktop 버튼을 클릭한 후 Github Desktop 프로그램을 설치한다.
    • Clone in Desktop 프로그램에 로그인
    • 다시 아까 Github사이트에서 자신의 리포지터리의 Clone in Desktop 버튼을 클릭한다.
    • 생성된 자신의 리포지터리에서 이 문서를 참조하여 sample1.html 을 생성한다.
    • Github 프로그램에서 변경된 파일을 Commit 하고 우측상단에 Sync 를 클릭합니다.
    • {username}.github.il/sample1 파일을 웹브라우져에서 확인한다.
